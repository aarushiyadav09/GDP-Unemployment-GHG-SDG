# GDP-Unemployment-GHG-SDG
An analysis of India's GDP, unemployment rates, and greenhouse gas emissions using data and the UN Sustainable Development Goals (SDG 8: Decent Work &amp; Economic Growth and SDG 13: Climate Action). includes projections for 2030, state-specific EDA, and ARIMA and Exponential Smoothing forecasts. Operations Research M.Sc., Aligarh Muslim University.
